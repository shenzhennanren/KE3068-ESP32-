# 7. 问题解答

1. kidsblock始终连接不上？

   - 请确保没有其他程序占用了通信串口。若Arduino IDE未完全关闭，可能导致串口资源被占用。
2. Arduino IDE 串口一直在跳动无法选择？
   - 请外接电源后再尝试。
2. Arduino IDE 上传代码时出现了 "uploading error: exit status 2" 的错误？
- 串口占用问题：请确保没有其他程序占用通信串口。
  
- 硬件问题：检查与主板连接的电缆和接线是否正确连接，并确保主板正常工作。
3. esp32 主板使用USB线连接到计算机，串口端口未显示？
   - 确保驱动已安装。点击“设备管理器”查看是否显示对应的串口端口。
   - 换一根USB线。
4. 烧录了代码，传感器/模块不工作或串口监视器窗口不能显示正确的信息？

   - 确保代码中的引脚和实际接线一致。
5. 舵机为什么工作不正常？

   - 确保舵机在安装前初始化，并且正确接线。

   - 请外接电源供电。
6. 无源蜂鸣器的音调与实际音调不准？

   - 普通无源蜂鸣器模拟的音调达不到专业的要求。
7. 人体红外热释电传感器误报？

   - 探测范围内避免强光照射干扰，如：阳光照射、汽车灯光照射、射灯照射及照明等光源。
8. AHT20温湿度传感器防水吗？

   - AHT20温湿度传感器检测的是空气中的温度和湿度，不防水，请勿将模块放入水中。
9. RFID模块读卡没反应？

   - 代码中的 UID 需要替换成你自己的。通过课程7.1可以读取你的 UID 码。
10. 安装软件后无法使用？

   - 为确保产品各部分的正常运行，建议选择与资料一致的软件及版本进行安装，使用最新版本可能存在与库文件等不兼容的情况导致产品无法正常使用。

11. 能否把所有代码一起烧录进去？

    - 不能，开发板一次只支持烧录一个代码。

12. WiFi一直连接不上？

    - 确保代码里的 WiFi 名称和密码已经替换为你的。
    - 确保你的 WiFi 网络是 2.4GHz 的，ESP32不支持 5GHz WiFi

13. 输入IP地址无页面？

    - 确保IP地址输入正确。
    - 检查手机/电脑是否与ESP32在同一网络。
